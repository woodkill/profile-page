# 개발자 웹 이력서 개발 로드맵

## 프로젝트 개요
HTML, CSS, JavaScript, Tailwind CSS를 이용하여 반응형 개발자 웹 이력서를 구축하는 프로젝트입니다.

---

## 📋 개발 단계

### Phase 1: 프로젝트 초기 설정
- [ ] 프로젝트 디렉토리 구조 설계
- [ ] HTML 기본 템플릿 생성 (`index.html`)
- [ ] Tailwind CSS 설정 및 CDN 연동
- [ ] 기본 메타 태그 및 viewport 설정

### Phase 2: HTML 구조 작성
- [ ] 헤더(Header) 섹션 마크업
  - 이름/직급 표시
  - 네비게이션 메뉴
- [ ] 소개(About) 섹션 마크업
  - 간단한 자기소개
  - 프로필 이미지
- [ ] 기술스택(Skills) 섹션 마크업
  - 기술 카드/배지 구조
- [ ] 경력사항(Experience) 섹션 마크업
  - 경력 타임라인 또는 리스트
- [ ] 프로젝트(Projects) 섹션 마크업
  - 프로젝트 카드 구조
- [ ] 푸터(Footer) 섹션 마크업
  - 연락처 정보
  - SNS 링크

### Phase 3: Tailwind CSS 스타일링
- [ ] 색상 및 폰트 시스템 정의
- [ ] 헤더 스타일링
  - 반응형 네비게이션
  - 홈 버튼/로고
- [ ] 소개 섹션 스타일링
  - 프로필 이미지 원형 처리
  - 텍스트 레이아웃
- [ ] 기술스택 섹션 스타일링
  - 카드/배지 디자인
  - 그리드 레이아웃
- [ ] 경력사항 섹션 스타일링
  - 타임라인 또는 카드 레이아웃
- [ ] 프로젝트 섹션 스타일링
  - 반응형 그리드
  - 호버 효과
- [ ] 푸터 스타일링
  - 연락처 아이콘
- [ ] 전체 반응형 디자인 점검 (모바일, 태블릿, 데스크탑)

### Phase 4: JavaScript 상호작용 기능
- [ ] 스크롤 네비게이션
  - 활성 섹션 하이라이트
  - Smooth scroll 구현
- [ ] 모바일 메뉴 토글 기능
- [ ] 라이트/다크 모드 기능
  - 테마 토글 버튼
  - localStorage 저장
- [ ] 섹션별 애니메이션
  - Scroll-trigger 애니메이션
  - 페이드인 효과

### Phase 5: 콘텐츠 채우기
- [ ] 개인 정보 입력
  - 이름, 직급
  - 자기소개 텍스트
- [ ] 기술스택 리스트 작성
  - Frontend: HTML, CSS, JavaScript, Tailwind CSS 등
  - Backend, DevOps 등 추가 기술
- [ ] 경력사항 입력
  - 회사명, 직급, 기간
  - 담당 역할 및 성과
- [ ] 프로젝트 정보 입력
  - 프로젝트명, 설명
  - 사용 기술, 링크

### Phase 6: 최적화 및 배포
- [ ] 성능 최적화
  - 이미지 최적화
  - 불필요한 CSS/JS 제거
- [ ] SEO 최적화
  - 메타 태그 작성
  - 구조화된 데이터(Schema)
- [ ] 브라우저 호환성 테스트
- [ ] 접근성(a11y) 점검
- [ ] 정적 호스팅 배포 (GitHub Pages, Vercel, Netlify 등)

## 📝 참고 사항

- **반응형 디자인**: 모바일 퍼스트(Mobile-First) 접근법 사용
- **성능**: 최소한의 JavaScript로 최대한의 효과 구현
- **유지보수성**: 간단한 구조로 추후 수정 용이하게 설계
- **접근성**: 웹 접근성 가이드라인(WCAG) 고려
